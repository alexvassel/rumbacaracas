Banner push down:

Recuerde siempre enviar los archivos .fla, .swf y las tipografías convertidas en curvas.

Cómo funciona

El formato Banner push down es una creatividad Flash que se muestra en 2 instancias: En la primera el anuncio se verá replegado, y tendrá un botón para poder abrirlo si se desea. En la última instancia el anuncio se visualizará desplegado empujando el contenido de la página hacia abajo y también tendrá el botón cerrar. La diferencia con el banner desplegable, es que en ésta el anuncio se abrirá o cerrará al seleccionar el botón. Si el usuario ingresa nuevamente al sitio verá el anuncio replegado.

Cómo cuenta

  • La primera vez que se visualice el anuncio, el mismo se va a encontrar replegado, contando la impresión.
  • Si el usuario decide desplegar el anuncio comenzará a contar el tiempo de visualización.
  • Si el banner llega al final de la animación, se registra la acción Anuncio completo.
  • La acción Desplegar y la acción Replegar se registrarán cada vez que el usuario despliegue o repliegue el anuncio respectivamente.

Cómo se programa

El formato se programa en una pieza en Flash, publicada en versión 6, 7, 8 ó 9. El mismo deberá tener en total dos frames: uno para el anuncio replegado y otro para el anuncio desplegado.

Medidas:

  • Banner replegado: 920x50 px
  • Banner desplegado: 920x200 px

Frame 1 "Anuncio replegado"

En el primer frame se debe llamar a la siguiente función de ActionScript:

_root.eplInicioAnuncio();

El banner se encuentra replegado con las medidas 920x50 px y sólo se despliega mediante el botón de "abrir" el cual debe incluir la siguiente acción:

on (release) {
_root.eplDesplegar();
_root.nextFrame();
}

Toda la animación que suceda en esta instancia deberá estar dentro de un MovieClip. Este será el MovieClip 1. La animación tendrá un loop infinito

Frame 2 "Anuncio desplegado"

Esta instancia se activa cuando el usuario decide abrir el banner. El banner se despliega con las medidas 920x200 px y solo se repliega mediante el botón de "cerrar" el cual debe incluir la siguiente acción:

on (release) {
_root.eplReplegar();
_root.nextFrame();
}

Toda la animación que suceda en esta instancia deberá estar dentro de un MovieClip. Este será el MovieClip 2. En el último frame de este MovieClip se debe llamar a la siguiente acción que permite que el anuncio se cierre solo:

_root.eplAnuncioCompleto();

En los dos frames debe haber un layer que contenga la acción de 

stop();
El área de click debe estar presente en los tres frames, y debe estar en un layer diferente. Llamando a la siguiente acción: 

getURL(_root.clickTag, "_blank");

Materiales a enviar

  • Un archivo .fla programado según las especificaciones anteriores
  • Un archivo .swf correspondiente al .fla
  • Recuerde que las tipografías deben estar convertidas en curvas dentro de la creatividad
